# Neural Network Playground

React + Vite playground showing that neural networks can learn relations, not just function graphs.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Shapes are loaded from the `svg/` directory.
- Training runs fully in the browser with TensorFlow.js.
- The UI follows the selected Split Lab layout from the project spec.
